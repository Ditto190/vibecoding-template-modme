# agents/

This directory holds **repo-specific** agent assets.  
Use assets from `vendor/awesome-copilot-index/` for anything reusable.

| Subdirectory | Purpose |
|---|---|
| `prompts/`   | Repo-specific prompt files |
| `review/`    | Repo-specific review checklists and instructions |
| `skills/`    | Repo-specific skill definitions |
| `subagents/` | Repo-specific subagent configurations |

If an asset here becomes broadly reusable, propose upstreaming it to
`Ditto190/awesome-copilot` per AGENTS.md § 3.
